# Japanese Stroke Mouse Writer V2.1.3 完全ガイド

[繁體中文](complete-guide.md) | [English](complete-guide.en.md) | 日本語

## 1. プログラムを開く

Portable ZIP 全体を展開し、`JapaneseStrokeMouseWriter.exe` をダブルクリックします。ZIP のプレビュー内や保護されたシステムフォルダーから実行しないでください。

## 2. 内容を入力してプレビューする

「内容とプレビュー」に日本語、英字、数字、対応記号を入力します。半角・全角スペースは1マス、Tab は4マス、Enter は新しい行または列になり、連続改行も保持されます。

英字は `A–Z` と `a–z`、数字は `0–9`、記号は `,， .． !！ ?？ :： ;； @＠ ~～ 、､ 。｡ ・･ ーｰ` に対応します。半角・全角記号はいずれも1マスを使用し、同じ筆順を共有します。収録済みの拗音、促音、小書き母音は正しい小さい字形を保ちます。それ以外の文字は書き込み前にエラーになります。薄い枠は文字位置、黒い線は実際の筆順経路です。

## 3. キャンバス範囲を検出する

1. 対象キャンバスを開き、鉛筆またはブラシを選びます。
2. 「開始座標を検出」を押します。
3. 右方向では左上、左方向では右上を指定します。
4. 「終了座標を検出」を押し、書き込み可能範囲の反対側を指定します。

検出中は ESC で取り消せます。座標は手入力でき、マルチモニターの負座標にも対応します。

## 4. 一般設定

- 文字サイズ：各文字マスの幅と高さ（px）。
- 文字間隔／行間隔：文字間および折り返し行・列の間隔。
- 配置方向：横書きまたは縦書き。
- 進行方向：右または左。

横書きは左右端で下へ折り返し、縦書きは下端で進行方向に新しい列を作ります。

### プリセット

名前付きレイアウトを追加、読込、上書き、名前変更、削除できます。文字サイズ、間隔、方向のみ保存し、文字と座標は保存しません。

## 5. 環境設定

- 言語：日本語、繁体字中国語、簡体字中国語、英語。即時に切り替わります。
- 開始カウントダウン：`0～30` 秒。
- 曲線精度：`0.1～20`。小さいほど滑らかですが時間がかかります。
- サンプル待機：`1～1000` ミリ秒。線が途切れる場合は増やします。

数値欄は全角数字と一般的な小数記号を自動変換します。

## 6. 書き込みの開始と停止

プレビューを確認して「書き始める」を押します。文字、資源、キャンバス範囲をすべて検証してからマウスが動きます。書き込み中は ESC、または画面の隅へマウスを移動して停止できます。

## 7. トラブルシューティング

- **非対応文字**：README に記載された日本語、英字、数字、記号を使用してください。
- **直線だけが描かれる**：直線・図形ツールではなく鉛筆またはブラシを選んでください。
- **キャンバス範囲外**：範囲を広げるか、文字サイズや間隔を小さくしてください。
- **フォルダーへ書き込めない**：プログラムフォルダー全体をデスクトップやドキュメントへ移動してください。
- **SmartScreen 警告**：V2.1.3 は未署名です。公式 Release から取得し SHA-256 を確認してください。
