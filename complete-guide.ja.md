# Japanese Stroke Mouse Writer V2.4.0 完全ガイド

[繁體中文](complete-guide.md) | [English](complete-guide.en.md) | 日本語

## 1. プログラムを開く

Portable ZIP 全体を展開し、`JapaneseStrokeMouseWriter.exe` をダブルクリックします。最大化状態で起動し、緊急停止案内は画面下部のステータスバーに表示されます。アプリ内の「使用説明」タブでも操作方法を確認できます。ZIP のプレビュー内や保護されたシステムフォルダーから実行しないでください。

## 2. 内容、顔文字、プレビュー

「内容とプレビュー」に日本語、英字、数字、対応記号、顔文字を入力します。半角スペースは `0.5` マス、全角スペースは `1` マス、Tab は半マス4個分の `2` マスです。Enter は新しい行または列になり、連続改行も保持されます。

左側には顔文字ピッカーがあります。笑／楽しい、かわいい、挨拶／謝罪、照れ、ラブ、悲しい／泣く、怒り、驚き、焦り／汗、眠い／疲れ、困惑／肩すくめ、動作／手振り、動物／キャラなどの分類から選び、文字欄へ挿入できます。内蔵対応の顔文字を手動で貼り付けることもできます。

顔文字はフォントアウトラインで描画され、黒いプレビュー線がそのままマウス出力経路になります。顔文字は1つのまとまりとして扱い、内部の字間はフォントの glyph advance に従います。半角／全角の字間規則は顔文字内部には適用せず、途中で自動改行しません。1つの顔文字が書き込み範囲に入らない場合は、マウスを動かす前にエラーになります。

英字と数字は半角・全角の `A–Z`、`a–z`、`0–9` に対応します。半角カタカナは `0.5` マスを使用し、`ｶﾞ`、`ﾊﾟ` などの濁音は1つの半マス字形になります。記号は `#＃ (（ )） [［ ]］ @＠ ~～ 、､ 。｡ ・･ ーｰ`、および `「」『』【】〈〉《》〔〕｢｣` に対応します。カラー絵文字、keycap、ZWJ シーケンスは非対応です。薄い枠は占有範囲、黒い線は実際の出力経路です。

## 3. キャンバス範囲を検出する

1. 対象キャンバスを開き、鉛筆またはブラシを選びます。
2. 「開始座標を検出」を押します。
3. 右方向では左上、左方向では右上を指定します。
4. 「終了座標を検出」を押し、書き込み可能範囲の反対側を指定します。

検出中は ESC で取り消せます。座標は手入力でき、マルチモニターの負座標にも対応します。

## 4. 一般設定

- 文字サイズ：各文字マスの幅と高さ（px）。
- 文字間隔／行間隔：文字間および折り返し行・列の間隔。
- 配置方向：横書きまたは縦書き。
- 進行方向：右または左。

半角文字同士は設定した文字間隔の半分を使用し、全角文字同士または半角と全角の組み合わせは完全な文字間隔を使用します。半角スペース、全角スペース、Tab にも同じ規則を適用します。横書きは左右端で下へ折り返し、縦書きは下端で進行方向に新しい列を作ります。縦書きでは英数字、括弧、長線を時計回りに回転し、`、。` をマスの右上へ移動します。顔文字は完全なアウトライン run として保持し、1文字ずつには分解しません。

### プリセット

名前付きレイアウトを追加、読込、上書き、名前変更、削除できます。文字サイズ、間隔、方向のみ保存し、文字と座標は保存しません。

## 5. 環境設定

- 言語：日本語、繁体字中国語、簡体字中国語、英語。即時に切り替わります。
- 開始カウントダウン：`0～30` 秒。
- 曲線精度：`0.1～20`。小さいほど滑らかですが時間がかかります。
- サンプル待機：`1～1000` ミリ秒。線が途切れる場合は増やします。

数値欄は全角数字と一般的な小数記号を自動変換します。

## 6. 書き込みの開始と停止

プレビューを確認して「書き始める」を押します。文字、顔文字、資源、キャンバス範囲をすべて検証してからマウスが動きます。書き込み中は ESC、または画面の隅へマウスを移動して停止できます。

## 7. トラブルシューティング

- **非対応文字**：README に記載された日本語、英字、数字、記号、内蔵顔文字を使用してください。
- **顔文字が範囲外になる**：範囲を広げるか、文字サイズを小さくするか、短い顔文字を選んでください。
- **直線だけが描かれる**：直線・図形ツールではなく鉛筆またはブラシを選んでください。
- **キャンバス範囲外**：範囲を広げるか、文字サイズや間隔を小さくしてください。
- **フォルダーへ書き込めない**：プログラムフォルダー全体をデスクトップやドキュメントへ移動してください。
- **SmartScreen 警告**：V2.4.0 は未署名です。GitHub Release から ZIP をダウンロードし、SHA-256 を確認してください。
