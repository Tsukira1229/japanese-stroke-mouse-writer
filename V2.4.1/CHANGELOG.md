# V2.4.1 Internal Changelog

- Removed the previous kaomoji category picker, built-in kaomoji preset data, kaomoji run matching, font fallback, and font-dependent drawing.
- Restored kaomoji input to normal character-by-character layout.
- Added project-authored centerline SVG resources for selected Unicode symbols useful when manually assembling kaomoji.
- Preserved existing Japanese, English, number, halfwidth katakana, common symbol, vertical rotation, and adaptive halfwidth/fullwidth spacing behavior.
- Kept SignPath status as not approved; this internal build remains unsigned and does not enable a signing workflow.
- Created an uncompressed internal portable output folder under `V2.4.1/`; no ZIP, tag, or GitHub Release is created for this version.
