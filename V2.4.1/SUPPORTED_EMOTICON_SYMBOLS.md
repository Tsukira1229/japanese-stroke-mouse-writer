# Supported Emoticon Symbols

V2.4.1 supports these additional centerline symbols for users who want to assemble kaomoji manually. The full SVG review checklist is available in [EMOTICON_SYMBOL_SVG_REVIEW_TABLE.md](EMOTICON_SYMBOL_SVG_REVIEW_TABLE.md), and a spreadsheet-friendly CSV is available in [emoticon-symbol-svg-review-table.csv](emoticon-symbol-svg-review-table.csv).

## Representative supported kaomoji

```text
(^O^)
(≧▽≦)
m(_ _)m
(/ω＼)
¯\_(ツ)_/¯
(╯°□°)╯︵ ┻━┻
ฅ^•ﻌ•^ฅ
(˶ᵔ ᵕ ᵔ˶)
ლ(ಠ益ಠ)ლ
ʕ•ᴥ•ʔ
ᕕ(ᐛ)ᕗ
(ㅠ﹏ㅠ)
```

These symbols are written as normal characters. They do not create a single protected kaomoji run, and they do not use font-dependent drawing. Unsupported emoji, keycap sequences, ZWJ sequences, and unsupported pictorial symbols are rejected before mouse movement.
