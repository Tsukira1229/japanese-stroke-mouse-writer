# Security policy

## Supported versions

Security fixes are provided for the latest published version.

## Reporting a vulnerability

Report vulnerabilities privately through [GitHub private vulnerability reporting](https://github.com/Tsukira1229/japanese-stroke-mouse-writer/security/advisories/new). Do not include credentials, API tokens, personal information, or exploit details in a public issue.

Please include the affected version, reproduction steps, expected impact, and any proposed mitigation. Reports will be acknowledged after review, and confirmed issues will be addressed before public disclosure when practical.
