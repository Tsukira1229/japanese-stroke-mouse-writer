# Japanese Stroke Mouse Writer V2.4.1 完整步驟

## 1. 輸入文字

在「內容與預覽」輸入或貼上要書寫的內容。空格、Tab、換行、連續換行與自動換行都會反映在結果中。顏文字請直接輸入文字框，例如 `(^O^)`、`(≧▽≦)`、`m(_ _)m`、`(/ω＼)`、`¯\_(ツ)_/¯`、`(╯°□°)╯︵ ┻━┻`。

V2.4.1 不提供顏文字分類插入，也不依賴字型外觀。顏文字會拆成逐字元中心線書寫，因此使用者可以自行用支援符號拼出需要的形狀。

## 2. 設定座標

起始座標與末端座標共同形成書寫矩形。向右時起始座標是左上角、末端座標是右下角；向左時起始座標是右上角、末端座標是左下角。可手動輸入 X、Y，也可使用偵測按鈕。偵測期間主視窗會最小化，倒數結束時讀取目前滑鼠座標；按 ESC 可取消。

## 3. 調整排版

支援範圍包含 `A–Z`、`a–z`、`0–9`、全形英數、半形片假名濁音如 `ｶﾞ`，以及常用半形／全形符號對應：`#＃`、`(（`、`)）`、`[［`、`]］`、`@＠`、`~～`、`、､`、`。｡`、`・･`、`ーｰ`、`「」`、`【】`、`｢｣`。

字體大小以 px 為單位。半形字元占 `0.5` 格，全形／寬字元占 `1` 格，Tab 等於四個半形空格；半形片假名濁音如 `ｶﾞ` 會合併為單一半格字元。半形對半形使用半字距；全形對全形或半形與全形混合時使用完整字距。水平文字碰到左右邊界後向下換行；垂直文字碰到底部後依流向新增左側或右側欄。

垂直時英數、括號與長線會順時針旋轉，`、。` 會移至字格右上。顏文字中的每個字元也遵循一般字元的直排規則。

## 4. 更新預覽

按「更新預覽」後，淡色字格代表占用範圍，黑色線條代表實際滑鼠書寫路徑。若出現不支援字元，程式會在移動滑鼠前提示。

## 5. 開始書寫與停止

確認預覽後按「開始書寫」。倒數期間切換到小畫家或其他畫布，並確認畫筆工具已選取。書寫期間可按 ESC 停止，也可將滑鼠移到螢幕角落觸發安全停止。程式停止或發生錯誤時會自動放開滑鼠左鍵。

## 6. 常見問題

- **不支援的書寫字元**：改用 README 或 `V2.4.1/SUPPORTED_EMOTICON_SYMBOLS.md` 列出的字元。
- **emoji 組合不支援**：彩色 emoji、keycap emoji 與 ZWJ 組合會在滑鼠移動前被拒絕。
- **顏文字形狀與期待不同**：V2.4.1 採逐字元中心線書寫，請調整符號組合、字距或使用全形／半形替換。
- **SmartScreen 警告**：V2.4.1 是未簽章內部開發版，可能出現未知發行者提示。
