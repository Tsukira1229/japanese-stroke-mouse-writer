# Privacy statement

Japanese Stroke Mouse Writer runs locally and does not collect telemetry, analytics, personal information, or usage data.

This program will not transfer any information to other networked systems unless specifically requested by the user or the person installing or operating it.

The Portable application reads bundled stroke data and writes settings only to `user_data/settings.json` beside the executable. The developer-only `install_kanjivg.py` and `scripts/generate_font_style_packs.py` scripts can download pinned KanjiVG or Google Fonts source files when explicitly run; neither script is used by the packaged application.
