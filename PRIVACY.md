# Privacy statement

Japanese Stroke Mouse Writer runs locally and does not collect telemetry, analytics, personal information, or usage data.

This program will not transfer any information to other networked systems unless specifically requested by the user or the person installing or operating it.

The Portable application reads bundled stroke data and writes settings only to `user_data/settings.json` beside the executable. The developer-only `install_kanjivg.py` script can download KanjiVG source data when explicitly run; it is not used by the packaged application.
