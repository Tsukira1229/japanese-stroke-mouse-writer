# 日文筆順滑鼠書寫工具：UI 使用教學

## 1. 第一次使用

先安裝 Python 3，安裝時勾選 **Add Python to PATH**。接著在本資料夾執行：

```powershell
python -m pip install -r requirements.txt
```

安裝只需要執行一次。KanjiVG 筆順資料已包含在 `data` 資料夾。

## 2. 開啟操作介面

雙擊 `run_writer.bat`。畫面左側可調整文字與書寫參數，右側顯示筆順預覽。

## 3. 輸入文字

在「書寫文字」輸入平假名、片假名或漢字，例如：

```text
こんにちは日本語
```

工具不會使用字型輪廓替代。若輸入非日文文字或缺少筆順資料，介面會直接顯示錯誤。

## 4. 設定書寫位置

1. 開啟小畫家並建立空白畫布。
2. 選擇鉛筆或筆刷，不要選直線、曲線或其他圖形工具。
3. 回到工具按「擷取滑鼠位置」。
4. 視窗最小化後，在 3 秒內把滑鼠停在第一個字左上角。

擷取完成後，介面會自動填入 X、Y 座標。

## 5. 調整參數

- **字寬／字高**：每個字使用的畫布大小。
- **字距／行距**：文字之間與換行後的距離。
- **開始倒數**：按下開始後保留的切換畫面時間。
- **取樣點停頓**：每個曲線點的等待時間，預設 `0.008`。
- **筆畫停頓**：每一筆之間的等待時間，預設 `0.03`。
- **曲線精細度**：數字越小越細緻，但書寫時間越長。

## 6. 預覽與書寫

按「產生預覽」，右側會顯示實際筆畫路徑；紅色數字表示筆畫開始順序。

確認後按「開始書寫」。介面會自動最小化，倒數結束後開始控制滑鼠。完成或發生錯誤時，介面會重新顯示。

## 7. 緊急停止

書寫期間若位置錯誤，快速將滑鼠移到整個螢幕左上角。程式會放開滑鼠按鍵並停止。

## 8. 常見問題

### 雙擊後沒有開啟

確認已安裝 Python 3，並重新執行：

```powershell
python -m pip install -r requirements.txt
```

### 只畫出直線

確認小畫家選擇的是鉛筆或筆刷，不是直線或圖形工具。

### 筆畫有斷線

把「取樣點停頓」提高到 `0.012`。

### 文字超出畫布

重新擷取書寫位置，或降低字寬、字高與字距。
