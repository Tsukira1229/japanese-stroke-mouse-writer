# 日文筆順滑鼠書寫工具 V1.0：使用教學

本工具專門書寫平假名、片假名與漢字。每個字都直接讀取 KanjiVG SVG 中的筆畫順序，再逐筆控制滑鼠；不會描繪字型外框。

## 1. 安裝套件與筆順資料

在本資料夾開啟 PowerShell：

```powershell
python -m pip install -r requirements.txt
python install_kanjivg.py
```

若 `data/kanjivg/20250816/main/kanji` 已存在，可略過第二個指令。重新安裝時，下載壓縮檔會在成功解壓後自動刪除。

## 2. 先檢查混合文字預覽

```powershell
python mouse_writer_pro.py --text "日本語かなカナ" --preview preview.png
```

預覽圖中的黑線是滑鼠實際移動路徑，紅色數字是全部筆畫的執行順序。這個步驟不會控制滑鼠。

## 3. 準備畫布

1. 開啟小畫家或其他繪圖程式。
2. 建立足夠大的空白畫布，選擇鉛筆或較細的筆刷。不要選擇直線、曲線或其他圖形工具，否則每次拖曳都只會得到起點到終點的線段。
3. 建議先使用 `100%` 縮放。
4. 確認書寫區不會被工具列、捲軸或其他視窗遮住。

程式使用整個螢幕的座標。移動畫布視窗、縮放或捲動畫布後，需要重新定位。

## 4. 取得書寫起點

```powershell
python locate_position.py --snapshot-after 3
```

倒數期間把滑鼠停在第一個字預定位置的左上角。記下輸出的 `X`、`Y`，例如：

```text
目前滑鼠座標：X=877, Y=325
```

## 5. 實際依筆順書寫

```powershell
python mouse_writer_pro.py --text "日本語かなカナ" --start-x 877 --start-y 325 --char-width 150 --char-height 150 --execute
```

按下 Enter 後會倒數 5 秒。倒數期間切到畫布視窗，之後不要碰滑鼠，直到終端機顯示完成。

緊急停止時，把滑鼠快速移到整個螢幕左上角；`pyautogui` 會中止程式並放開滑鼠按鍵。

## 6. 常用調整

縮小文字與字距：

```powershell
python mouse_writer_pro.py --text "かなカナ日本" --char-width 100 --char-height 100 --char-gap 8 --start-x 300 --start-y 300 --execute
```

線條漏畫或斷裂時降低速度：

```powershell
python mouse_writer_pro.py --text "かなカナ日本" --point-delay 0.012 --start-x 300 --start-y 300 --execute
```

換行書寫：

```powershell
python mouse_writer_pro.py --text "ひらがな`nカタカナ`n日本語" --preview preview.png
```

## 7. 問題排除

### 缺少 Python 套件

```powershell
python -m pip install -r requirements.txt
```

### 找不到 KanjiVG 筆順資料

```powershell
python install_kanjivg.py --force
```

若只有特定字元報錯，代表目前資料版本沒有該字的筆順檔。工具會直接停止，不會改用輪廓或其他字型代替。

### 文字超出畫布

重新取得座標，或降低 `--char-width`、`--char-height` 和 `--char-gap`。

### 筆畫斷線

先使用較細的筆刷，再提高到 `--point-delay 0.012`。若仍有漏點，可再加上 `--move-duration 0.002`。
