# Japanese Stroke Mouse Writer V2.0.2 Portable 使用教學

## 1. 開啟程式

完整解壓 Portable ZIP，雙擊 `JapaneseStrokeMouseWriter.exe`。不要直接在 ZIP 預覽中執行，也不要將程式放在不可寫入的系統資料夾。

## 2. 內容與預覽

在「內容與預覽」輸入要書寫的日文。文字框不會自行改變換行，因此下列內容都會反映到實際排版：

- 一般空格與全形空格：各占一個字格。
- Tab：占四個字格。
- Enter：建立新的行或欄。
- 連續 Enter：保留空白行或空白欄。

右側淡色格線只用來顯示排版位置；黑色線條才是實際滑鼠書寫路徑。文字框不顯示滾動條，但仍可使用滑鼠滾輪、Shift + 滾輪及鍵盤捲動。

## 3. 偵測畫布範圍

1. 開啟小畫家，建立空白畫布並選擇鉛筆或筆刷。
2. 按「偵測起始座標」。
3. 主視窗最小化後，在 3 秒內移動滑鼠：向右流向選擇可寫範圍左上角，向左流向選擇可寫範圍右上角。
4. 按「偵測末端座標」，將滑鼠移到可寫範圍的另一端。

偵測過程可按 ESC 取消，原座標不會被覆寫。X、Y 欄位也可直接輸入，並支援多螢幕負座標。

## 4. 一般設定

- **字體大小**：KanjiVG 字格的寬與高，單位 px。
- **字距**：同一行或欄的字元間距。
- **行距**：換行或換欄間距。
- **排列方向**：水平或垂直。
- **流向**：向右或向左。

排版規則：

- 水平向右／向左：碰到左右邊界後向下換行。
- 垂直向左／向右：字元由上往下，碰到底部後向左或向右換欄。

### 自訂選項

可新增、載入、覆寫、重新命名與刪除多個自訂選項。自訂選項只保存字體大小、字距、行距、排列方向與流向，不保存文字及座標。

## 5. 環境設定

- **語言**：依序提供日本語、繁體中文、简体中文與 English；選擇後立即切換並自動保存。首次啟動依 Windows 顯示語言選擇。
- **開始倒數**：按下開始後切換到畫布的時間，0–30 秒。
- **曲線精細度**：範圍 `0.1–20`；越小越細緻，但書寫時間越長。
- **取樣點停頓**：範圍 `1–1000` 毫秒，預設 `8` 毫秒；筆畫斷線時可適度提高。

三個數值欄位可直接輸入或使用上下箭頭；全形數字及 `。`、`．`、`，`、`,` 會自動轉成標準數字格式。環境設定會自動保存到 Portable 資料夾。

## 6. 開始書寫與停止

確認預覽後按「開始書寫」。程式會先檢查所有文字是否能放入畫布矩形；任何字元超界時都不會移動滑鼠。

書寫期間按 ESC 可全域停止，即使主視窗已最小化也有效。也可將滑鼠快速移到任何螢幕角落觸發第二層 failsafe。

## 7. 問題排除

### Portable 資料夾無法寫入

將整個程式資料夾移到桌面、文件或其他一般使用者可寫入的位置。

### 只畫出直線

確認畫布選擇的是鉛筆或筆刷，不是直線、曲線或圖形工具。

### 預覽顯示超出畫布

增加末端範圍、縮小字體，或降低字距／行距。

### Windows SmartScreen 顯示警告

V2.0.2 是未簽章的開源 Portable 版本。請只從專案官方 GitHub Release 下載並核對 SHA-256。
