# Japanese Stroke Mouse Writer V2.3.2 完整步驟

繁體中文 | [English](complete-guide.en.md) | [日本語](complete-guide.ja.md)

## 1. 開啟程式

完整解壓 Portable ZIP 後，雙擊 `JapaneseStrokeMouseWriter.exe`。程式會以最大化視窗啟動，緊急停止提示顯示於視窗底部狀態列；程式內的「使用說明」頁籤也包含完整操作方式。不要直接在 ZIP 預覽中執行，也不要放在不可寫入的系統資料夾。

## 2. 輸入內容與預覽

在「內容與預覽」輸入日文、英文字母、數字或支援的符號。文字框不會自行改變換行：一般空格占 `0.5` 格、全形空格占 `1` 格、Tab 顯示四個半格並占 `2` 格；Enter 建立新行或新欄，連續 Enter 會保留空白行或欄。

英文支援半形／全形 `A–Z`、`a–z`，數字支援半形／全形 `0–9`。半形片假名占 `0.5` 格，`ｶﾞ`、`ﾊﾟ` 等濁音組合會合併為單一半格字形。符號包含 `#＃ (（ )） [［ ]］ @＠ ~～ 、､ 。｡ ・･ ーｰ`，另支援 `「」『』【】〈〉《》〔〕｢｣`。Unicode 半形／窄字元占 `0.5` 格，全形／寬字元占 `1` 格；對應字元共用筆順。拗音、促音、小母音等內附小假名會依正確的小字比例顯示。其他字元會在書寫前顯示錯誤。右側淡色格線是實際占格位置，黑色線條是筆順路徑。

## 3. 偵測畫布範圍

1. 開啟目標畫布並選擇鉛筆或筆刷。
2. 按「偵測起始座標」。
3. 向右流向選擇可寫範圍左上角；向左流向選擇右上角。
4. 按「偵測末端座標」，選擇可寫範圍另一端。

倒數期間可按 ESC 取消。座標可手動輸入，並支援多螢幕負座標。

## 4. 一般設定

- 字體大小：每個字格的寬高，單位 px。
- 字距／行距：字格間距及換行／換欄間距。
- 排列方向：水平或垂直。
- 流向：向右或向左。

半形字元彼此相鄰時使用設定字距的一半；全形對全形或半形與全形混合時使用完整字距。一般空格、全形空格及 Tab 亦遵循此規則。水平文字碰到左右邊界後向下換行；垂直文字碰到底部後依流向新增左側或右側欄。垂直時英數、括號與長線會順時針旋轉，`、。` 會移至字格右上。

### 自訂選項

可新增、載入、覆寫、重新命名與刪除多個排版選項。只保存字體大小、字距、行距、方向與流向，不保存文字和座標。

## 5. 環境設定

- 語言：日本語、繁體中文、简体中文、English，選擇後立即切換。
- 開始倒數：`0–30` 秒。
- 曲線精細度：`0.1–20`，越小越細緻但耗時更長。
- 取樣點停頓：`1–1000` 毫秒，筆畫斷線時可提高。

數字欄位支援全形數字及常見小數符號自動轉換。

## 6. 開始與停止書寫

確認預覽後按「開始書寫」。程式會先完成所有字元、資源及畫布範圍檢查；任何錯誤都不會移動滑鼠。書寫期間可按 ESC，或將滑鼠移到螢幕角落停止。

## 7. 問題排除

- **不支援的書寫字元**：改用 README 列出的日文、英文字母、數字或符號。
- **只畫出直線**：確認目標工具是鉛筆或筆刷，不是直線或圖形工具。
- **超出畫布**：增加範圍、縮小字體或降低字距／行距。
- **資料夾無法寫入**：將整個程式資料夾移到桌面、文件或其他可寫入位置。
- **SmartScreen 警告**：V2.3.2 未簽章，請從官方 Release 下載並核對 SHA-256。
