# Japanese Stroke Mouse Writer V2.7.1 Development

Windows 10/11 x64 免安裝滑鼠書寫工具。程式會將日文、英數字與支援符號轉換成中心線筆跡，並在小畫家或其他可使用滑鼠繪圖的畫布中書寫。

[English](README.en.md) / [日本語](README.ja.md)

## 安裝方式

1. 將完整的 `JapaneseStrokeMouseWriter-v2.7.1-development-win-x64-portable` 資料夾放到可寫入的位置，例如文件或桌面。
2. 不要只移動 EXE；`_internal`、`user_data` 與其他隨附檔案必須保留在原本相對位置。
3. 執行 `JapaneseStrokeMouseWriter.exe`。程式不需要安裝 Python，也不需要系統管理員權限。
4. 若 Windows SmartScreen 顯示未知發行者，請先確認檔案來自本專案，再選擇「其他資訊」與「仍要執行」。目前程式未簽章。

設定會保存到程式資料夾內的 `user_data/settings.json`，不使用 Registry 或 AppData。請將程式放在目前帳號具有寫入權限的位置。

## 快速開始

1. 開啟小畫家或其他目標畫布，選好畫筆或鉛筆工具。
2. 在程式的「內容與預覽」輸入文字。
3. 使用「偵測起始座標」及「偵測末端座標」指定可書寫矩形。
4. 在「一般設定」選擇KanjiVG、Yomogi、Zen Kurenaido或Hachi Maru Pop書寫風格，並調整字體大小、字距、行距、排列方向與流向。
5. 按「更新預覽」，確認所有黑色筆跡都位於矩形內。
6. 按「開始書寫」，在倒數期間切換回目標畫布。

完整操作方式與座標方向說明請直接開啟[HTML 完整說明](complete-guide.html)。GitHub 上也提供 [Markdown 版本](complete-guide.md)。

## 功能

- 可選擇KanjiVG (預設)、Yomogi、Zen Kurenaido或Hachi Maru Pop；預覽與實際滑鼠輸出使用同一份路徑。
- 保留空格、Tab、換行與連續換行，碰到書寫矩形邊界時自動換行或換欄。
- 支援水平、垂直、向右與向左排版；預覽與實際滑鼠路徑使用相同資料。
- 半形字元占 `0.5` 格，全形／寬字元占 `1` 格；半形對半形使用半字距，其餘組合使用完整字距。
- 可將字體大小、字距、行距、排列方向、流向與書寫風格保存為多個自訂選項。
- 提供亮色主題、暗色主題與跟隨 Windows；首次使用預設為亮色主題，切換外觀時會保留目前輸入及操作狀態。
- 可直接輸入顏文字，例如 `(^O^)`、`(≧▽≦)`、`m(_ _)m`、`(/ω＼)` 與 `(╯°□°)╯︵ ┻━┻`。程式會逐字使用支援符號的中心線書寫。

## 支援文字

支援日文假名、KanjiVG 收錄的漢字、`A–Z`、`a–z`、`0–9`、全形英數及半形片假名。`ｶﾞ`、`ﾊﾟ` 等合法濁音／半濁音組合會合併成單一半格字元。

Yomogi風格包含6,608個中心線SVG（另有96個目錄字元回退KanjiVG），並包含`ゟ`、`ヿ`兩個Yomogi專屬假名。Zen Kurenaido包含6,591個中心線SVG、111個回退字元；Hachi Maru Pop包含6,608個中心線SVG、94個回退字元。三款字型的可用目錄字元分別有6,606、6,591、6,608份已鎖定的最佳努力繪製順序；每條原始骨架線段都會完整使用一次，但不保證正統日文筆順。順序資料缺少、損壞或與來源字形不符時，程式會安全回退原本的中心線路徑順序。

支援可列印 ASCII 標點及全形對應，例如 `#＃`、`(（`、`)）`、`[［`、`]］`、`@＠`、`~～`；日文符號包含 `、､`、`。｡`、`・･`、`ーｰ`、`「」`、`【】`、`｢｣`。

Unicode 框線區段 `U+2500–U+257F` 的 128 個字元全部可用。要讓 `┏━┷━┓` 等相鄰框線連續，請將字距與行距設為 `0 px`。

程式支援一部分常用特殊符號，例如幾何、星形、勾叉、箭頭、括號及數學符號：`☆★`、`○●`、`△▲`、`◇◆`、`□■`、`✓✔`。特殊符號並非全部可用；完整清單見 [支援中心線符號](SUPPORTED_SYMBOLS.md)，未支援字元會在開始書寫前提示。

## 緊急停止

座標偵測、開始倒數或實際書寫期間都可按 `ESC` 停止。書寫時也可將滑鼠快速移到任一螢幕角落觸發安全停止；程式會放開滑鼠左鍵。
