# Japanese Stroke Mouse Writer V2.4.0 Portable

[繁體中文](README.md) | [English](README.en.md) | 日本語

Windows 10/11 x64 向けのポータブル書き込みツールです。Windows SendInput を使用し、日本語、英数字、一般記号に加えて、フォントアウトラインで形を正確に描く顔文字を書き込めます。

## インストール方法

1. GitHub Release から `JapaneseStrokeMouseWriter-v2.4.0-win-x64-portable.zip` をダウンロードします。
2. ZIP 全体を書き込み可能なフォルダーへ展開します。
3. `JapaneseStrokeMouseWriter.exe` をダブルクリックします。

Python や管理者権限は不要です。インストーラー、アンインストーラー項目、ショートカットは作成しません。設定は実行ファイル横の `user_data/settings.json` に保存されます。

## 対応内容

- 日本語：ひらがな、カタカナ、半角カタカナ、KanjiVG に含まれる漢字。`ｶﾞ`、`ﾊﾟ` などは1つの半角字形として扱います。
- 小書き仮名：拗音、促音、小母音などの小さいひらがな／カタカナは小さい比率を保ちます。
- 英字：半角 `A–Z`、`a–z`、全角 `Ａ–Ｚ`、`ａ–ｚ`。
- 数字：半角 `0–9`、全角 `０–９`。
- ASCII 記号と全角対応：<code>!！ "＂ #＃ $＄ %％ &＆ '＇ (（ )） *＊ +＋ ,， -－ .． /／ :： ;； &lt;＜ =＝ &gt;＞ ?？ @＠ [［ \\＼ ]］ ^＾ _＿ `｀ {｛ |｜ }｝ ~～</code>。
- 日本語の句読点と括弧：`、､ 。｡ ・･ ーｰ 「」 『』 【】 〈〉 《》 〔〕 ｢｣`。
- 顔文字：笑／楽しい、かわいい、挨拶／謝罪、照れ、ラブ、悲しい／泣く、怒り、驚き、焦り／汗、眠い／疲れ、困惑／肩すくめ、動作／手振り、動物／キャラの分類を内蔵しています。
- 顔文字はフォントアウトラインで描画され、黒いプレビュー線と実際のマウス出力が一致します。顔文字は1つのまとまりとして扱い、途中で自動改行しません。
- Unicode の半角／狭幅文字は `0.5` マス、全角／広幅文字は `1` マスを占有します。対応する半角／全角は同じ筆順を共有します。
- 通常スペースは `0.5` マス、全角スペースは `1` マス、Tab は4つの半角マスとして表示され `2` マスを占有します。改行は保持されます。
- 半角文字同士は設定した文字間隔の半分を使います。どちらかが全角文字の場合は完全な文字間隔を使います。

非対応文字、筆順データのない文字、カラー絵文字、keycap、ZWJ シーケンスは、マウスを動かす前に拒否されます。

## 主な機能

- 横書き／縦書き、右方向／左方向のレイアウト。
- 縦書きでは英数字、括弧、長音記号を自動回転し、`、。` をマス右上へ移動します。
- 開始／終了座標の検出、自動折り返し、範囲外チェック。
- プレビューとマウス出力は同じレイアウト経路を共有します。
- 「内容とプレビュー」ページから分類別に顔文字を挿入できます。
- 複数の名前付きレイアウトプリセット。
- 日本語、繁體中文、简体中文、English のデスクトップ UI。
- ESC のグローバル緊急停止と画面隅の failsafe。
- 起動時の最大化、下部ステータスバーの緊急停止ヒント、内蔵の「使用説明」タブ。

操作方法は [完全ガイド](complete-guide.ja.md) を参照してください。
