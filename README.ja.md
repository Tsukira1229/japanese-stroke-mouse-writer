# Japanese Stroke Mouse Writer V2.7.1

Windows 10/11 x64 向けのポータブルなマウス書き込みツールです。日本語、英数字、対応記号を中心線の筆跡へ変換し、Paint などのマウス操作対応キャンバスへ書き込みます。

[繁體中文](README.md) / [English](README.en.md)

## インストール方法

1. `JapaneseStrokeMouseWriter-v2.7.1-win-x64-portable.zip` をダウンロードして展開し、フォルダー全体をドキュメントやデスクトップなど書き込み可能な場所へ置きます。
2. EXE だけを移動しないでください。`_internal`、`user_data`、その他の同梱ファイルは元の相対位置に保持します。
3. `JapaneseStrokeMouseWriter.exe` を実行します。Python と管理者権限は不要です。
4. Windows SmartScreen が発行元不明の警告を表示した場合は、本プロジェクトから取得したファイルであることを確認してから「詳細情報」「実行」を選択してください。現在のプログラムは未署名です。

設定はプログラムフォルダー内の `user_data/settings.json` に保存されます。Registry と AppData は使用しないため、現在のアカウントで書き込める場所を使用してください。

## クイックスタート

1. Paint などの対象キャンバスを開き、ペンまたは鉛筆ツールを選択します。
2. 「内容とプレビュー」に文字を入力します。
3. 「開始座標を検出」と「終了座標を検出」で書き込み範囲を指定します。
4. 「一般設定」でKanjiVG、Yomogi、Zen Kurenaido、Hachi Maru Popのいずれかを選択し、文字サイズ、文字間隔、行間隔、方向、流れを調整します。
5. 「プレビュー更新」を押し、黒い書き込み経路が範囲内にあることを確認します。
6. 「書き始める」を押し、カウントダウン中に対象キャンバスへ切り替えます。

座標方向を含む詳しい操作は、オフラインで開ける [HTML 完全ガイド](complete-guide.ja.html)を参照してください。GitHub では [Markdown 版](complete-guide.ja.md)も利用できます。

## 機能

- KanjiVG (既定)、Yomogi、Zen Kurenaido、Hachi Maru Popを選択でき、プレビューとマウス出力は同じパスを使用します。
- スペース、Tab、改行、連続改行を保持し、書き込み範囲の端で自動的に折り返します。
- 横書き、縦書き、右向き、左向きに対応し、プレビューとマウス出力は同じ経路データを使用します。
- 半角文字は `0.5` マス、全角／幅広文字は `1` マスです。半角同士は半分の文字間隔、それ以外は完全な文字間隔を使います。
- 文字サイズ、文字間隔、行間隔、方向、流れ、書き込みスタイルを複数の名前付きプリセットとして保存できます。
- ライト、ダーク、Windows に合わせる外観モードを選択できます。初回はライトテーマで開始し、外観を切り替えても入力内容と操作状態を保持します。
- `(^O^)`、`(≧▽≦)`、`m(_ _)m`、`(/ω＼)`、`(╯°□°)╯︵ ┻━┻` などの顔文字を直接入力できます。各対応文字を中心線で書き込みます。

## 対応文字

日本語の仮名、KanjiVG に収録された漢字、`A–Z`、`a–z`、`0–9`、全角英数字、半角カタカナに対応します。`ｶﾞ`、`ﾊﾟ` など正しい濁音／半濁音の組み合わせは、1つの半角文字として扱います。

Yomogiには6,608個の中心線SVGと96個のフォールバックがあり、専用仮名`ゟ`、`ヿ`も含みます。Zen Kurenaidoには6,591個の中心線SVGと111個のフォールバック、Hachi Maru Popには6,608個の中心線SVGと94個のフォールバックがあります。対応するカタログ文字には、それぞれ6,606、6,591、6,608件の固定済みベストエフォート描画順があります。元の骨格線分はすべて1回ずつ使用されますが、正統な日本語の筆順を保証するものではありません。順序データが欠落、破損、または元の字形と一致しない場合は、元の中心線パス順へ安全にフォールバックします。

表示可能な ASCII 記号と全角対応を使用できます。例：`#＃`、`(（`、`)）`、`[［`、`]］`、`@＠`、`~～`。日本語記号は `、､`、`。｡`、`・･`、`ーｰ`、`「」`、`【】`、`｢｣` を含みます。

Unicode 罫線素片 `U+2500–U+257F` の全128文字に対応します。`┏━┷━┓` のように隣接セルを接続する場合は、文字間隔と行間隔を `0 px` に設定してください。

幾何、星、チェック、矢印、括弧、数学など、一部の一般的な特殊記号に対応します。例：`☆★`、`○●`、`△▲`、`◇◆`、`□■`、`✓✔`。すべての特殊記号が使用できるわけではありません。完全な一覧は[対応中心線記号](SUPPORTED_SYMBOLS.md)を参照してください。未対応文字は書き込み開始前に通知されます。

## 今後の予定

- 中心線で書きやすい特殊記号を継続して追加します。

## 緊急停止

座標検出、開始カウントダウン、書き込み中は `ESC` で停止できます。書き込み中にマウスを画面の隅へ素早く移動しても安全停止します。停止時はマウス左ボタンを解放します。
