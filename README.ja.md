# Japanese Stroke Mouse Writer V2.3.0 Portable

[繁體中文](README.md) | [English](README.en.md) | 日本語

Windows 10/11 x64 向けのインストール不要な筆順書き込みツールです。Windows SendInput を使用し、ペイントなどのキャンバスへ日本語、英字、数字、記号を中心線の筆順に沿って書き込みます。

## インストール方法

1. `JapaneseStrokeMouseWriter-v2.3.0-win-x64-portable.zip` をダウンロードします。
2. ZIP 全体を書き込み可能なフォルダーへ展開します。
3. `JapaneseStrokeMouseWriter.exe` をダブルクリックします。

Python や管理者権限は不要です。インストーラーやアンインストール項目は作成されません。設定は実行ファイル横の `user_data/settings.json` に保存されます。

## 対応文字

- 日本語：ひらがな、カタカナ、半角カタカナ、KanjiVG に収録された漢字。`ｶﾞ`、`ﾊﾟ` などは1つの半マス字形になります。
- 小書き仮名：拗音、促音、小書き母音など、収録済みの小書き平仮名・片仮名は小さい字形を保ちます。
- 英字：半角 `A–Z`、`a–z`、全角 `Ａ–Ｚ`、`ａ–ｚ`。
- 数字：半角 `0–9`、全角 `０–９`。
- ASCII 記号と全角対応：<code>!！ "＂ #＃ $＄ %％ &＆ '＇ (（ )） *＊ +＋ ,， -－ .． /／ :： ;； &lt;＜ =＝ &gt;＞ ?？ @＠ [［ \\＼ ]］ ^＾ _＿ `｀ {｛ |｜ }｝ ~～</code>。
- 日本語記号・括弧：`、､ 。｡ ・･ ーｰ 「」 『』 【】 〈〉 《》 〔〕 ｢｣`。
- Unicode の半角／狭い文字は `0.5` マス、全角／広い文字は `1` マスを使用し、対応する字形は同じ筆順データを共有します。
- 半角スペースは `0.5` マス、全角スペースは `1` マス、Tab は半マス4個分の `2` マスです。改行も保持されます。

非対応文字や筆順データがない文字は、マウスを動かす前にエラーになります。

## 主な機能

- 横書き／縦書き、左方向／右方向の配置。
- 縦書きでは英数字、括弧、長線を自動回転し、`、。` をマスの右上へ移動します。
- 開始・終了座標の検出、自動折り返し、範囲チェック。
- プレビューとマウス出力で同じレイアウト経路を使用。
- 複数の名前付きレイアウトプリセット。
- 日本語、繁体字中国語、簡体字中国語、英語の画面表示。
- ESC による緊急停止と画面隅のフェイルセーフ。
- 起動時に最大化し、画面下部のステータスバーに緊急停止案内を表示します。アプリ内に「使用説明」タブもあります。

操作方法は [完全ガイド](complete-guide.ja.md) を参照してください。
