# Japanese Stroke Mouse Writer V2.4.0

V2.4.0 replaces the earlier local special-symbol experiment with a released kaomoji-focused portable build.

## Changes

- Added curated kaomoji categories in the Content & Preview page.
- Added font-outline drawing for built-in kaomoji so the black preview paths match mouse output.
- Kept kaomoji as indivisible runs; they do not auto-wrap in the middle.
- Preserved existing KanjiVG centerline writing for Japanese, kana, kanji, English, numbers, common symbols, halfwidth katakana, small kana, vertical rotation, and adaptive halfwidth/fullwidth spacing.
- Removed the previous local V2.4.0 special-symbol experiment and generated SVG resources.
- Updated SignPath documentation: the Foundation application was not approved at this time, so V2.4.0 remains unsigned.
