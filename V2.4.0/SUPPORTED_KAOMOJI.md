# V2.4.0 Supported Kaomoji

The built-in kaomoji list is curated for outline drawing stability. Each kaomoji is treated as one run and is not split by automatic wrapping.

## Categories

- Happy: `(^O^)`, `(≧▽≦)`, `(*´▽｀*)`, ``ヽ(=´▽`=)ﾉ``
- Cute: `(๑•﹏•)`, `(｡◕‿◕｡)`, ``(❁´ω`❁)``, `٩꒰｡•◡•｡꒱۶`
- Greeting / apology: `(^-^)/`, `m(_ _)m`, `＼(^o^)／`, `ヾ(＾∇＾)`
- Shy: `(/ω＼)`, `(//∇//)`, `(*ﾉωﾉ)`, `(〃▽〃)`
- Love: `(づ｡◕‿‿◕｡)づ`, `(人❛ᴗ❛)`, `(♡˙︶˙♡)`, `(*˘︶˘*).｡.:*♡`
- Sad / crying: `(T_T)`, `(；ω；)`, `(｡•́︿•̀｡)`, `(╥﹏╥)`
- Angry: `(｀Д´)`, `(╬ಠ益ಠ)`, `ヽ(｀Д´)ﾉ`, ``(＃`Д´)``
- Surprised: `(ﾟ∀ﾟ)`, `(⊙ꇴ⊙)`, `Σ(ﾟДﾟ)`, `(°□°)`
- Anxious / sweat: `(^_^;)`, `(；´Д｀)`, `(汗)`, `(・_・;)`
- Sleepy / tired: `(-_-)zzz`, `(＿ ＿*) Z z z`, `(∪｡∪)｡｡｡zzz`, ``(´-ω-`)``
- Confused / shrug: `¯\_(ツ)_/¯`, `┐(´∀｀)┌`, `╮(╯_╰)╭`, `┐(´д｀)┌`
- Action / gesture: `(╯°□°)╯︵ ┻━┻`, `o(≧▽≦)o`, `٩(ˊᗜˋ*)و`, `∑d(°∀°d)`
- Animal / character: `ฅ^•ﻌ•^ฅ`, `(·(ｪ)·)`, `(▽◕ ᴥ ◕▽)`, `⊂((〃￣⊥￣〃))⊃`

Color emoji, keycap sequences, and ZWJ sequences are not supported.
