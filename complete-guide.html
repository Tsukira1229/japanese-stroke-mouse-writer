Japanese Stroke Mouse Writer V2.7.0 Development 完整步驟

本指南從第一次開啟程式開始，說明如何設定書寫範圍、確認預覽並安全執行。

1. 開啟程式與準備畫布

  1. 執行 JapaneseStrokeMouseWriter.exe。程式預設以最大化視窗開啟，介面語言會依 Windows 語言選擇，也可在「環境設定」切換。
  2. 開啟小畫家或其他可用滑鼠繪圖的程式。
  3. 在目標程式選擇畫筆或鉛筆工具，設定顏色與筆刷粗細。
  4. 確認畫布已足夠大，且沒有對話框或其他視窗遮住預定書寫位置。

2. 輸入書寫內容

在「內容與預覽」輸入或貼上文字。空格、Tab、換行與連續換行都會保留；未手動換行的內容會在碰到書寫矩形主方向邊界時自動換行或換欄。

可直接輸入顏文字，例如 (^O^)、(≧▽≦)、m(_ _)m、(/ω＼) 與 (╯°□°)╯︵ ┻━┻。顏文字由各個支援字元的中心線依序組成，因此符號選擇、半形／全形及字距都會影響結果。

3. 設定起始與末端座標

起始座標與末端座標會形成可書寫矩形，座標是滑鼠在整個 Windows 桌面上的 X、Y 位置，多螢幕環境可能出現負數。

  • 水平或垂直、流向向右：起始座標是矩形左上角，末端座標是右下角。
  • 水平或垂直、流向向左：起始座標是矩形右上角，末端座標是左下角。

建議使用偵測功能：

  1. 按「偵測起始座標」。主視窗會最小化，游標所在螢幕會顯示十字線、即時 X/Y 座標與 3 秒倒數。
  2. 將十字線交叉點移到目標畫布對應的上方角落，保持不動直到程式取得座標。游標跨到另一個螢幕時，十字線會自動移到該螢幕。
  3. 按「偵測末端座標」，再將滑鼠移到對應的下方角落。已取得的起點會固定顯示；起點與游標在同一螢幕時，畫面也會即時顯示預計書寫矩形。若位於不同螢幕，起點所在螢幕仍會保留起點準星與座標。
  4. 程式恢復後確認 X、Y 已填入。也可以直接輸入座標。

偵測期間按 ESC 會取消，原本座標不會被覆蓋。矩形至少需要容納一個字格；範圍或方向不合法時，程式會在預覽前提示。

4. 調整一般設定

  • 字體大小：每個全形字格的寬與高，單位為 px。
  • 字距：相鄰字元的距離。半形對半形使用設定值的一半，其他組合使用完整字距。
  • 行距：水平排版的行距或垂直排版的欄距。
  • 排列方向：水平時逐字左右排列；垂直時逐字向下排列。
  • 流向：向右或向左，並決定起始座標使用矩形哪一側。
  • 書寫風格：KanjiVG原始筆跡遵循原資料筆順；Yomogi、Zen Kurenaido與Hachi Maru Pop直繪中心線直接重現各字型骨架，路徑順序不代表傳統筆順。

半形字元占 0.5 格，全形／寬字元占 1 格，Tab 等於四個半形空格。ｶﾞ、ﾊﾟ 等合法半形片假名組合占一個半格。

垂直排版時，英數字、括號及長線會順時針旋轉；、。 會移到字格右上。箭頭與框線保留原方向。

「自訂選項」可新增、載入、覆寫、重新命名或刪除設定。自訂選項保存字體大小、字距、行距、排列方向、流向與書寫風格，不保存文字及座標。

5. 調整環境設定

  • 語言：日文、繁體中文、簡體中文或英文。
  • 外觀主題：可選擇亮色主題、暗色主題或跟隨 Windows。首次使用預設為亮色主題。
  • 開始倒數：按下開始後，保留給使用者切換到目標畫布的秒數，範圍 0–30 秒。
  • 曲線精細度：範圍 0.1–20；數字越小，曲線越細緻，但產生的點越多、書寫時間越長。
  • 取樣點停頓：每個曲線點的等待時間，範圍 1–1000 毫秒。若筆畫斷線，可逐步提高。

環境設定會自動保存。切換語言或外觀時，輸入文字、座標、目前頁籤及視窗狀態都會保留。一般情況可先使用預設值，只在曲線不夠平滑或筆畫斷線時調整。

6. 確認支援範圍

支援日文假名、KanjiVG 收錄的漢字、A–Z、a–z、0–9、全形英數、半形片假名與常用符號。

Yomogi提供6,608個直繪字元與96個回退字元，並包含ゟ、ヿ兩個無KanjiVG對照的專屬假名。Zen Kurenaido提供6,591個直繪字元與111個回退字元；Hachi Maru Pop提供6,608個直繪字元與94個回退字元。回退僅限各風格包明確列出的來源缺字或未通過轉換門檻字元。

半形／全形符號包括 #＃、(（、)）、[［、]］、@＠、~～、、､、。｡、・･、ーｰ、「」、【】、｢｣。程式另支援一部分常用的幾何、星形、勾叉、箭頭、括號、數學及框線符號，完整清單見 支援中心線符號。

特殊符號並非全部可用。若文字中含有未支援的特殊符號，程式會在開始書寫前指出第一個不支援字元。

輸入框線時，將字距與行距設為 0 px，即可連接 ┏━┷━┓、┃　　┃、┗━━━┛ 等相鄰端點。

7. 更新預覽

按「更新預覽」。淡色格線代表每個字元占用的範圍，黑色線條就是實際送到滑鼠的書寫路徑。

開始前請確認：

  • 所有內容都位於書寫矩形內。
  • 空格、換行與自動換行位置符合預期。
  • 實心符號的外框與疏斜線符合需要。
  • 沒有不支援字元或範圍不足提示。

8. 開始書寫與緊急停止

  1. 按「開始書寫」。
  2. 在開始倒數期間切換到目標畫布，不要再移動目標視窗或改變畫布縮放。
  3. 倒數結束後，程式會從起始座標依預覽路徑控制滑鼠書寫。

座標偵測、開始倒數或書寫期間都可按 ESC 停止。書寫時也可將滑鼠快速移到任一螢幕角落觸發安全停止。正常完成、停止或錯誤時，程式都會放開滑鼠左鍵。

9. 常見問題

  • SmartScreen 顯示未知發行者：目前程式未簽章。請只執行由本專案取得且內容完整的資料夾。
  • 無法保存設定：將整個程式資料夾移到文件或桌面等可寫入位置。
  • 筆畫斷線：提高取樣點停頓，並避免使用過快或不穩定的目標筆刷。
  • 曲線不夠平滑：降低曲線精細度；數值越低，書寫時間越長。
  • 結果位置偏移：重新偵測座標，並確認倒數後沒有移動畫布、視窗或改變縮放。
  • 顏文字比例不理想：調整字距，並確認使用的是預期的半形或全形符號。
